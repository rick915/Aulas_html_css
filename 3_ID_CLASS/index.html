ID CLASS

ID

Id funciona como cpf Assim como so posso ter um numero de CPF, cada elemento so pode ter um valor de ID. O CPF é unico, então não é possivel repetir o mesmo valor duas vezes.

Class

O CLASS funciona de maneira oposta ao id. Difirente do ID em que você so pode ter um valor, no CLASS você pode ter vsarios valores. Tambem diferente ID, em que não se pode repetir um valor, no CLASS voce pode utilizar o mesmovalor para varios elementos.
